## Landing page React

Site one-page présentant Agentic AI (Hero, About, Features, Insights, Contact).

**Stack :** React 19 · Vite · Tailwind CSS 4 · Lucide React · Bootstrap Icons

### Démarrage rapide

```bash
cd front_end-frameworks/react
npm install
npm run dev
```

L’app est disponible sur [http://localhost:3000](http://localhost:3000).

Pour plus de détails (scripts, structure, déploiement), voir le [README du projet React](./front_end-frameworks/react/README.md).
